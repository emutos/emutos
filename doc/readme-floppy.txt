EmuTOS - floppy version

This special version allows EmuTOS to be loaded from a boot floppy.

emutos.st - single sided floppy disk image

This version is compatible with any Atari hardware (except the FireBee)
or emulator.
Due to size limitation, the only included language is English.

Be sure to remove the EmuTOS floppy when asked for.

This special version has been built using:
make flop
